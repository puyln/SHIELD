# Notices

This project contains original SHIELD project structure and local generation code prepared for public release.

The implementation is designed to interoperate with external research code bases, including DINOv2 and MONAI SwinUNETR. Those projects are not vendored in this repository. Users should consult the upstream repositories and licenses before using or redistributing their code or model weights.

Clinical datasets, imaging files, labels, and trained checkpoints are not included.
